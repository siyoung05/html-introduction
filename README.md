# 자기소개
 
 ## 사용한 언어
 ### HTML5 + CSS3
 
 ## 코드 설명
 ● CSS를 사용하여 각 태그에 맞는 스타일을 지정해줌.
 
 ● section태그를 이용하여 6가지 형식으로 나누었으며 각 section마다 id값을 넣어주어 하이퍼링크를 사용할 수 있도록 해주었다.
 
 ● nav태그를 사용하여 a와href를 이용해 section으로 빠르게 이동할 수 있게 해주었다.
 
 ● footer와 &copy를 이용해 영문이름과 주소를 입력함.
