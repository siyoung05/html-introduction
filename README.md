# 자기소개서
 
 ## 사용 언어
 html
 
 ## 코드 설명
 (1) section태그를 이용하여 6가지 형식으로 나누었으며 각 section마다 id값을 넣어주어 하이퍼링크를 사용할 수 있도록 해주었다.
 
 (2) nav태그를 사용하여 a와href를 이용해 section으로 빠르게 이동할 수 있게 해주었다.
 
 (3) footer와 &copy를 이용해 사이트를 만들면서 이용한 책의 이름을 적음.
